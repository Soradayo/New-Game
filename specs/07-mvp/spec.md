# MVP

## 概要

MVPは、最初に完成させるプレイ可能版を定義します。

## 要件

MVPには必ず次を含めます:

- full life playthrough
- 5 actions
- 5 stances
- relationships
- procedural NPCs
- weighted events
- event logs
- save/load
- JSON export/import
- offline support
- mod loading

おおよその範囲:

- 50 events
- 20 items

## 設計

非目標:

- multiplayer
- graphics
- animation
- runtime AI generation
- scripting mods
- complex simulation realism
