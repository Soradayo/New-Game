# UI

## 概要

UIはテキストファーストです。

キャラクターグラフィックは意図的に除外します。

## 要件

UIは必ず次を表示します:

- character information
- stats
- event logs
- actions
- relationships

## 設計

上部:

- name
- age
- class
- affiliation
- money

左:

- abilities

中央:

- logs

下部:

- actions
- stance

右:

- relationships

タブ:

- history
- relationships
- organization
- inventory
- world state
