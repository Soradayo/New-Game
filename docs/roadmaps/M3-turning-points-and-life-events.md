# M3: 転機と人生イベント

## Summary

人生の方向を変える転機と、それに接続する個人史イベントを実装する。M3では「選択した人生が違えばログと状態が明確に違う」状態を作る。

重要な各種イベントのうち、個人史・進路・関係性に関わるものはこの段階で実装する。

## Key Changes

- 転機カテゴリを整理する。
  - `education`
  - `career`
  - `relationship`
  - `organization`
  - `world`
- 進学、就職、所属、思想、家族・親密関係など、人生の方向を変える転機を追加する。
- 転機には選択条件、選択不可理由、結果説明、付与タグ、NPC影響、世界影響を持たせる。
- 転機前後に接続するイベントを追加する。
  - 個人史イベント。
  - 関係性イベント。
  - 進学・就職・所属に関わるイベント。
  - 転機の予兆や後日談。

## Content Target

- turningPoints: 10〜12件。
- personal / relationship / career events: 40〜50件。

## Completion Criteria

- 進学、就職、所属、関係性の主要な転機が一通り発生する。
- 選択肢によって、経歴タグ、教育、職業、関係、ログが変わる。
- 転機前後の通常イベントが、選択した人生に応じて変化する。
- M3終了時点で複数周回の人生差分が読める。

## Test Plan

- 転機カテゴリごとの条件判定。
- 選択不可理由の表示。
- choice適用による経歴タグ、教育、職業、NPC状態、世界状態の変化。
- 転機前後イベントが該当タグ・状態で出現すること。
- 追加イベントのschema validation。

## Assumptions

- 世界情勢そのものを大きく変動させるイベントはM5で扱う。
- M3のworldカテゴリ転機は、個人の人生分岐に関係するものに限定する。
- コンテンツ量はM6でさらに増補する。
