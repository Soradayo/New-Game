# M5: 世界シミュレーションと世界イベント

## Summary

世界を大きく変動させ、同じ人物でも時代と世界情勢が違えば人生が変わる状態を作る。M5では世界が単なる背景ではなく、人生を曲げる外力として働くことを目標にする。

重要な各種イベントのうち、社会変動・組織勢力・景気・弾圧・宗教・労働運動などはこの段階で実装する。

## Key Changes

- 世界状態を拡張する。
  - `era`
  - `organizationInfluence`
  - `unrest`
  - `economy`
  - `faith`
  - `repression`
  - world tags
- 国家・組織・社会圧が、人生の進路、職業、所属、NPCの運命に影響する。
- 世界値が一定閾値を超えた場合、大きな世界イベントが発火する。
- 世界イベントはプレイヤー、NPC、組織、後続イベント条件に影響する。
- 世界変化は構造化ログと世界タブに表示する。

## Content Target

- world events: 30〜40件。
- organization events: 15〜20件。

## Completion Criteria

- 世界値がターンやイベントで変動する。
- 世界値やworld tagsを条件にしたイベントが発火する。
- 世界イベントがプレイヤーとNPCの状態に影響する。
- 世界タブで現在の世界情勢を読める。

## Test Plan

- world condition/effectのunit test。
- 世界値の閾値によるイベント発火。
- 世界イベントによるプレイヤー、NPC、組織状態の変化。
- 世界ログの構造化。
- world schema validation。

## Assumptions

- 世界は大きく変動させるが、主役は個人の人生と関係性。
- 世界シミュレーションは数値の厳密性より、人生への因果を優先する。
- 戦争や内乱相当の表現は架空世界の抽象イベントとして扱う。
